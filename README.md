# WalkTaiwan TDX觀光網頁

## DEMO
網頁預覽： [DEMO](https://yhes9604.github.io/WalkTaiwain/#/)

## 使用技術
* 使用 Vue.js / Vue Vite 架設，Vue-Router 管理路由
* 使用 Vue axios 串接TDX API 連接資料
* 引入 Bootstrap5，製作 RWD 響應式網頁
* 使用 ESLint 管控程式撰寫風格


## 功能介紹

### 前台
* 首頁串接TDX資料，隨機顯示推薦景點
* 點擊每個活動都可以查看詳細資料
* 頁面都有google map地圖顯示位置
* 可以篩選縣市活動、景點、餐廳

##  網頁操作
1. 預設首頁 
2. 首頁隨機推薦活動、景點、餐廳
3. 網頁圖片有預設圖片
